### Node server
```
npm run start
```
### Vue client
```
npm run serve
```
### Docker compose build with no cache
```
docker-compose build --no-cache
```
### Docker start frontend & backend 
```
docker-compose up
```
### Docker compose build & start image/container
```
docker-compose up --build 
```

### Compiles and minifies for production
```
npm run build
```
